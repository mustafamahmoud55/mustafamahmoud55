### 🧠 About Me

I'm a **Node.js Backend Developer** specializing in production-grade, cloud-native systems — from UGC marketplaces and SaaS platforms to multi-domain enterprise ERP backends.

- 🏗️ I architect **scalable REST APIs** with complex data models, multi-role access control, and payment gateway integrations
- ⚡ I build **real-time systems** using Socket.IO — live messaging, event-driven notifications, and multiplayer backends
- 🗄️ I optimize **databases** across MongoDB, PostgreSQL, and MySQL — cutting API response times by 35–40% through indexing and query design
- ☁️ I integrate **cloud-native infrastructure** — Cloudflare R2 direct uploads, GitHub Actions CI/CD, Docker, and PM2 deployments
- 💳 I engineer **payment flows** with Paymob Checkout & Payouts (EGP/SAR), HMAC webhooks, and idempotent fulfillment logic
